Mail Gateway
============

.. i18n: E-mail gateways can automatize and update cases in the CRM via e-mails. Users can

.. i18n: is dedicated to allow users to assign cases.

E-mail gateways can automatize and update cases in the CRM via e-mails. Users can
create, assign or update cases to the CRM simply by sending or receiving e-mails
without going into OpenERP interface. This module is a very useful tool
to manage all from generic e-mails from the company (info, sales, support,...) and
is dedicated to allow users to assign cases.
