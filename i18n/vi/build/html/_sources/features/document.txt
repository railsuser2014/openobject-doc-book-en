Document Management System
==========================

.. i18n: The document management system allows you to store, classify, and search your
.. i18n: enterprise's documents. This module provides an FTP access to all your ERP's
.. i18n: resources.  For a dynamic and efficient classification, the folders structure
.. i18n: is provided by other objects.  It is very useful to track documents related
.. i18n: to projects, products, sales, contracts, etc.

The document management system allows you to store, classify, and search your
enterprise's documents. This module provides an FTP access to all your ERP's
resources.  For a dynamic and efficient classification, the folders structure
is provided by other objects.  It is very useful to track documents related
to projects, products, sales, contracts, etc.
