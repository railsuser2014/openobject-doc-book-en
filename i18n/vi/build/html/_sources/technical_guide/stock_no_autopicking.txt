.. i18n: Description
.. i18n: -----------

Description
-----------

.. i18n: ::
.. i18n: 
.. i18n:   This module allows an intermediate picking process to provide raw materials
.. i18n:       to production orders.
.. i18n:   
.. i18n:       One example of usage of this module is to manage production made by your
.. i18n:       suppliers (sub-contracting). To achieve this, set the assembled product
.. i18n:       which is sub-contracted to "No Auto-Picking" and put the location of the
.. i18n:       supplier in the routing of the assembly operation.

::

  This module allows an intermediate picking process to provide raw materials
      to production orders.
  
      One example of usage of this module is to manage production made by your
      suppliers (sub-contracting). To achieve this, set the assembled product
      which is sub-contracted to "No Auto-Picking" and put the location of the
      supplier in the routing of the assembly operation.
