This repository is used to manage Open ERP features. All features are encoded
in .RST files, designed to be used by Sphinx. We will create one feature file
for each big Open ERP features. A feature may describe several Open ERP modules,
they are just used to provide a quick overview of main Open ERP features to
customers.

All features share the same description template:

Title
-----

10 lines of description in one or two paragraphs.

Key Points:

* Key point 1
* Key point 2

Integration Benefits:

* Benefit 1
* Benefit 2
