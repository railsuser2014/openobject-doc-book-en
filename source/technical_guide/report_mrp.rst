MRP Management - Reporting (*report_mrp*)
=========================================
:Module: report_mrp
:Name: MRP Management - Reporting
:Version: 5.0.1.0

:Web: http://www.openerp.com

Description
-----------

::

  A module that adds new reports based on MRP cases.
      Workcenter loads, Weekly Stock value variation
