
Module Hotel Management (*hotel*)
=================================
:Module: hotel
:Name: Hotel Management
:Version: 5.0.1.0
:Directory: hotel
:Web: 

Description
-----------

::

  Module for Hotel/Resort/Property management. You can manage:
      * Configure Property
      * Hotel Configuration
      * Check In, Check out
      * Manage Folio
      * Payment
  
      Different reports are also provided, mainly for hotel statistics.

Dependencies
------------

 * base - installed
 * product - installed
 * sale - installed
